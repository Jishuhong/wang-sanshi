## 部署到互联网

要将网站部署到互联网以获得可分享的链接，有以下选项：

### 选项 1：GitHub Pages（推荐）
1. 访问 https://github.com/Jishuhong/wang-sanshi/settings/pages
2. 在 "Source" 下选择 "Deploy from branch"
3. 分支选择 "main"，文件夹选择 "/totng-site"
4. 点击 Save，等待部署完成
5. 你的网站将在 `https://jishuhong.github.io/wang-sanshi/totng-site/` 上线

### 选项 2：Netlify Drop
1. 访问 https://app.netlify.com/drop
2. 将 `totng-site` 文件夹拖放到页面上
3. 获得一个临时链接（可注册账号绑定自定义域名）

### 选项 3：Vercel
1. 访问 https://vercel.com/new
2. 导入 GitHub 仓库 `Jishuhong/wang-sanshi`
3. 设置根目录为 `totng-site`
4. 点击 Deploy
